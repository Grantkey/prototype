Web-Based Client

All transactions can be performed in the browser making interaction with the network fast, simple, and secure. The web client automatically generates a key pair on the client-side which is stored locally and can be used to interact with the blockchain. Store the private key in a secure location.

Smart-Contracts

Smart-contracts can be used to transfer tokens based on the criteria specified in the contract. All transactions are automatically validated by and signed by the authority and contract validation is immediate.

Verified Immutable Transactions

The authority publishes signed transactions to a blockchain allowing any user to audit the veracity of the network.

Open Source

A trusted open source code base that can be audited for vulnerabilities.

About

Grantkey achieves instantaneous contract clearing while maintaining cryptographically secured immutability by moving from decentralized processing / validation to authority-based processing with decentralized validation. The authority is prevented from cheating because it signs every block meaning that if it ever attempts to tamper with the blockchain anyone with a receipt of a block signed by the authority that is not present in the blockchain will be able to prove that authority tampered with the chain and a new authority can be selected. This project was created as part of my masters research at NYU for CS-GY 9963 Advanced Project in Computer Science. The intent is to provide a functional prototype of the architecture described in my research. It is my hope that others will build on this research and either fork this project or create their own implementations.
